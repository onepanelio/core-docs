Overview

You can install Onepanel on:

  • Public cloud
  • Single node cluster on desktop, edge or cloud

#Installing on public cloud

Follow these guides to install on single node cluster or desktop.

  • To install on Azure Kubernetes Engine (AKS) follow AKS installation guide
  • To install on Google Kubernetes Engine (GKE) follow GKE installation guide
  • To install on Elastic Kubernetes Service (EKS) follow EKS installation guide

#Installing on a single node cluster

Follow these guides to install on a single-node cluster or your desktop.

  • To install on MicroK8s follow MicroK8s installation guide
