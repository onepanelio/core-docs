Overview

You can install Onepanel on:

  • Public cloud
  • Single node cluster on desktop, edge or cloud

Installing on public cloud#

Follow these guides to install Onepanel in public cloud.

  • To deploy on Azure Kubernetes Engine (AKS) follow AKS deployment guide
  • To deploy on Google Kubernetes Engine (GKE) follow GKE deployment guide
  • To deploy on Elastic Kubernetes Service (EKS) follow EKS deployment guide

Installing on a single-node cluster#

  • To deploy on Minikube follow Minikube deployment guide
  • To deploy on MicroK8s follow MicroK8s deployment guide
